# 双人麻将游戏

基于 Vue 3 + TypeScript 的纯前端双人麻将游戏。

## 游戏规则

1. **基础规则**：遵从日麻规则，包括点数计算
2. **发牌**：玩家先摸牌13张，第一轮随机庄家
3. **连庄规则**：
   - 庄家自摸则连庄
   - 流局时庄家宣告听牌连庄
   - 否则轮庄
4. **听牌机制**：
   - 玩家听牌后需要宣告听牌
   - 宣告后，另一人本局内不得再宣告听牌
5. **猜牌机制**：
   - 宣告听牌后，另一人可选2张牌猜测对方胡的牌
   - 如果任意一张猜中，本局结束
   - 否则听牌者摸4张牌
   - 如果自摸，计算点数，本轮结束
   - 否则另一人重新猜测，如此反复直到流局

## 技术栈

- Vue 3 (Composition API)
- TypeScript
- Vite
- Pinia (状态管理)

## 项目结构

```
src/
├── components/          # Vue 组件
│   ├── GameBoard.vue   # 游戏主面板
│   ├── PlayerHand.vue  # 玩家手牌显示
│   └── TileComponent.vue # 单个麻将牌组件
├── stores/             # Pinia 状态管理
│   └── gameStore.ts    # 游戏状态
├── types/              # TypeScript 类型定义

├── utils/              # 工具函数
│   ├── tiles.ts        # 牌相关工具（洗牌、发牌、听牌判断等）
│   └── scoring.ts      # 日麻计分逻辑
├── App.vue
└── main.ts
```

## 运行项目

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 游戏操作

1. 点击"开始游戏"按钮开始新游戏
2. 点击"摸牌"按钮从牌山摸一张牌
3. 点击手牌中的牌选中，然后点击"打牌"按钮打出
4. 当手牌符合听牌条件时，可点击"宣告听牌"
5. 对方宣告听牌后，选择2张牌进行猜测
6. 游戏结束后可选择"下一局"或"重新开始"

## 功能特点

- ✅ 完整的日麻牌型（136张牌）
- ✅ 听牌判断算法
- ✅ 日麻役种识别（立直、断幺九、平和、七对子、混一色、清一色等）
- ✅ 点数计算系统
- ✅ 独特的猜牌机制
- ✅ 连庄/轮庄逻辑
- ✅ 美观的UI界面

## 注意事项

- 本游戏为简化版日麻，部分复杂役种未实现
- 听牌判断使用简化算法，可能存在边界情况
- 纯前端实现，无需后端服务器
